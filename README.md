# Surify Browser Extension

## Getting Started

First, run the development server:

```bash
# Install dependencies
pnpm install
# Run web server
pnpm dev
# or
npm run dev
```

## Making production build

Run the following:

```bash
pnpm build
# or
npm run build
```

## Making package as zip file

Run the following:

```bash
pnpm run package
```

You can find the package zip file in the build folder.
